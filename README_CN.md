# 功能特性

- **KernelSU**：面向 Android GKI 设备的 Root 解决方案，以内核模式运行，直接在核心层授予用户应用 Root 权限。
- **SUSFS**：专为 KernelSU 设计的 Root 隐藏内核补丁及用户空间模块。
